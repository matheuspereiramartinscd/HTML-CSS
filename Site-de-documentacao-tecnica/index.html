Introdução

HTML é um acrônimo para HyperText Markup Language, e é o linguagem padrão para criar páginas da web. Cada página da web em que você navega seu navegador é construído usando HTML. O principal uso de HTML para fornecer uma estrutura adequada à página web para que todo o conteúdo ou dados do página poderia representar adequadamente. Um HTML autônomo pode criar apenas páginas estáticas e com aparência de esqueleto em preto e branco, mas com a ajuda de CSS e JavaScript, podemos criar um ambiente mais interativo e página web intuitiva. Quando tentamos visitar um site ou clicar no botão link, basicamente solicitamos ao servidor que nos mostre a página, então o servidor age de acordo com nossa solicitação e nos envia um documento HTML apropriado como resposta. Em seguida, esse documento HTML foi analisado por navegação e podemos para ver o conteúdo.

HTML Document

!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>      
<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
</body>
</html>

HTML Elements

Os elementos HTML fornecem o significado semântico para a página da web contente. Normalmente, usamos de forma intercambiável os termos elementos HTML e tags, mas tecnicamente ambos são diferentes. Uma tag HTML é apenas um caractere dentro do colchete <>, enquanto o elemento HTML é um coleção de tag inicial, seu atributo, conteúdo e tag final. Para exemplo:

<p class= "para"> Hello World </p>

HTML Attributes

Em HTML, os atributos são usados ​​para fornecer informações adicionais sobre os elementos. Para a maioria dos elementos HTML, os atributos são opcional, mas existem alguns elementos em que temos que entregar o atributos. Atributos sempre especificados na tag de abertura de o elemento HTML e eles especificados em um par de nome e valor. Exemplo:

<image src= "cat.jpg" alt ="cat image"> >

Neste exemplo, src ="cat.jpg" e alt="cat image" são dois atributos onde src e alt são nomes de atributos e "cat.jpg" e "imagem de gato" são valores de atributos. Aqui o atributo alt é opcional, mas src é obrigatório porque src especifica qual imagem mostrar. Lá deve haver pelo menos um intervalo de espaço entre dois atributos, e o valor dos atributos deve ter residido no duplo invertido vírgula.

HTML Heading

Para exibir o título da seção, título ou subtítulo, podemos usar o Marcas de cabeçalho HTML. No HTML 5, temos 6 tags de cabeçalho começando em < h1> até < h6>, onde < h1> especifica o cabeçalho maior e < h6> representam o menor ou subtítulo. Se o conteúdo for especificado por tags de cabeçalho, ele seria exibido grande e em negrito como em comparação com outros conteúdos de texto presentes na página da web. Exemplo: 

<h1>First Heading </h1>
<h2>Second Heading </h2>
<h3>Third Heading </h3>
<h4>Forth Heading </h4>
<h5>Fifth Heading </h5>
<h6>Sixth Heading </h6>

HTML Paragraph

Em HTML, os parágrafos podem ser definidos usando o elemento < /p> < p> . Parágrafo o texto sempre começa a partir de uma nova linha, o navegador analisou a tag < p> e adicione automaticamente alguma margem e espaço em branco após o final < /p> etiqueta. 

<p> Hello! and Welcome to the site! </p>
          <p> Here you get to know all about the latest technology. </p>

HTML Style

Cada navegador tem um mecanismo específico que analisa o documento HTML e exibe um estilo padrão do conteúdo da página. 

<body style="background-color:yellow;">

<h1>Website</h1>
<p>Welcome to the Website</p>
          
</body>

HTML Formatting

Em HTML, temos muitos elementos especiais que podem fornecer significado ao conteúdo do texto. 

HTML Elements	Description
<b>	Bold the text
<strong> An alternative for <b>, which tell that this text is essential.
<i>	Italic the text
<em>	Similar to italic but used when we want to emphasize the text.
<mark>	It marks the text with a default "yellow" background colour
<small>	Decrease the text size
<del>	It prints a cross line over the text.
<ins>	Represents the inserted text by putting an underline.
<sub>	This element is used to display the subscript.
<sup>	It can make a text superscript.
